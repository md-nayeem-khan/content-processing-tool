\textbf{Question 2:} Assume that soft-throttling is used with a rate of 100 requests per minute and a 5\% exceed in the limit is allowed. The client has sent the 104th request within the time limit. However, no resources are available to serve the request. What should the system do with this incoming request?
\\[0.2cm]
\begin{itemize}
 \item Discard the request and send an error response to the client.
 \item \textbf{[CORRECT]} Add a request to the queue and wait for the resource to become available.
\\[0.1cm]
\textit{Explanation:} 
Since with 5\% exceed in the limit, the rate limiter can allow 105 requests per minute. Therefore, the request will be allowed and the system will keep the request in the queue until the required resource becomes available.
\end{itemize}
